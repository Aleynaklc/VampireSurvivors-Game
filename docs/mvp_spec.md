# MVP Spec

## Amac

Ilk MVP'nin amaci para kazanmak degil, cekirdek oynanis hissini dogrulamaktir.

Basari sorusu:

`Oyuncu bir run bitince ikinci run'i hemen baslatmak istiyor mu?`

## MVP Icinde Olanlar

### Core Systems

- Tek arena
- Snake hareketi
- Govde buyumesi
- Kafa temasi ile yuksek hasar
- Govde temasi ile dusuk surekli hasar
- Kuyruk tehlike bolgesi
- Can sistemi
- Wave tabanli run
- Tek boss
- Seviye atlama ve 3 secenekli guc draft
- 12 gucluk ilk havuz
- Run sonu ozet ekrani

### Power Families

- `Body-Tech`
- `Fire`
- `Electric`
- `Critical/Frenzy`

## MVP Disinda Olanlar

- Gercek meta progression agaci
- IAP
- Reklam entegrasyonu
- Leaderboard
- Ikinci biome
- Kozmetik sistem
- Gunluk gorevler
- Battle pass
- LiveOps

- `Wave 1`: 90 saniye
- `Draft 1`
- `Wave 2`: 90 saniye
- `Draft 2`
- `Wave 3`: 90 saniye
- `Boss`
- `Summary`

## Duygu Hedefleri

- Ilk 15 saniye: kontrol rahatligi
- Ilk 45 saniye: guclenme sinyali
- Ilk 90 saniye: build sahibi oldugunu hissetme
- Run sonu: near-miss veya mastery hissi

## Kabul Kriterleri

1. Mobil input ile yilan kontrolu net hissedilmeli
2. Uzayan govde okunabilir kalmali
3. En az 3 farkli build hissi olusmali
4. Boss, build farkini hissettirmeli
5. Test oyuncusu run sonu nedeni anlayabilmeli: "niye oldum" sorusu bos kalmamali

## Ilk Playtest Sorulari

- Kontrol sinir bozucu muydu?
- Guc secimleri ilginc geldi mi?
- Yilan uzadikca oyun daha iyi mi oldu, daha kotu mu?
- Boss adil hissettirdi mi?
- Tekrar denemek ister misin?
