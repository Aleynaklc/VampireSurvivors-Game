# Product Brief

## Oyun Tanimi

`Snake Roguelite`, oyuncunun bir yilani surukleyerek arena icinde hayatta kaldigi, dusmanlari kafa carpmasi ve govde temasi ile erittigi, run boyunca guc draft ettigi bir hybrid-casual aksiyon oyunudur.

## Tek Cumlelik Farklilasma

`Silah atesi yok; yilanin uzunlugu hem saldiri hem savunma kaynagidir.`

## Hedef Oyuncu

- `Hybrid-casual` aksiyon oyunlarini seven mobil oyuncular
- `Vampire Survivors` benzeri build cesitliligini seven ama daha kisa session isteyen kitle
- Kisa oturumlarda net guclenme ve near-miss hissi arayan kullanicilar

## Tasarim Hedefleri

1. Ilk 10 saniyede anlasilan kontrol
2. Ilk 60 saniyede gorulur bir guclenme
3. Run sonunda "bir daha" dedirten near-miss veya build meraki
4. Tek kisilik ekip tarafindan uretilip genisletilebilir kapsam

## Core Loop

1. Arena'ya gir
2. Dusman ve pickup topla
3. Seviye atla, 3 guc arasindan sec
4. Govdeyi uzat, sinerji kur
5. Wave temizle, boss'u gec
6. Run odulunu al
7. Yeni run icin farkli build dene

## Meta Loop

1. Run tamamla veya fail ol
2. Kalici token kazan
3. Yeni baslangic secenegi, relic veya guc ailesi ac
4. Yeni biome ve dusman tipi gor
5. Farkli build yolu denemek icin geri don

## Fun Pillars

### 1. Coil Mastery

Oyuncu sadece kacmaz; govdesini konumlandirarak alan kontrolu kurar.

### 2. Controlled Chaos

Dogru guc kombinasyonu kuruldugunda ekran kaotik gorunur ama oyuncu nedeni anlar.

### 3. Risky Growth

Uzamak guclenmedir ama kuyrugu korumayi zorlastirir.

## Ilk "Wow Moment"

45-60. saniyede ilk buyuk sinerji devreye girer:

- Govde izi hasar birakir
- Kafa temasi zincirleme etki tetikler
- Kisa surede cok sayida dusman erir
- Ekran shake, sayi burst ve ses bunun sonucunu hissettirir

## Session Hedefi

- Ortalama run: `4-6 dakika`
- Ilk seans: `2-3 run`
- Tek run icinde: `3 wave + 1 boss`

## Monetization Ilkeleri

- Flow'u bozan agresif reklam yok
- Once retention, sonra monetization optimizasyonu
- Rewarded ad once run sonu bonus chest uzerinden denenir
- Interstitial ancak retention zarar gormezse test edilir

## Basarisizlik Nedenleri

- Snake hissinin zayif kalmasi
- Power draft'larin jenerik hissettirmesi
- Run icinde anlasilir guclenme olmamasi
- Fazla survivor-klon hissi

## Ilk Uretim Karari

`Merge` ve `idle` sistemleri MVP disindadir. Once `snake combat + power draft + short run satisfaction` dogrulanir.
