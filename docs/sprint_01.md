# Sprint 01

Sure: `5 is gunu`

Hedef:

`Snake combat hissi ve tekrar oynama istegi ureten cekirdek run loop'unu cikarmak.`

## Gun 1

- Unity projesi ac
- `MainPrototype` scene olustur
- Drag tabanli snake movement
- Govde takip mantigi

Teslim:

- Oyuncu yilanin kafasini rahat yonlendirebiliyor olmali

## Gun 2

- Kafa hasari ve govde temas hasari
- Kuyruk zayif bolge isareti
- Basit dusman `Chaser`
- Can sistemi

Teslim:

- Oyuncu riskli kafa vurusunu ve guvenli govde eritmesini hissedebilmeli

## Gun 3

- XP pickup
- Level up
- 3 secenekli draft popup
- Ilk 6 guc entegre et

Teslim:

- Guc seciminin run'i aninda degistirdigi gorulmeli

## Gun 4

- Wave director
- Elite dusman denemesi
- Ilk boss prototipi
- Run end screen

Teslim:

- 4-6 dakikalik tam run akisi oynanabilir olmali

## Gun 5

- Juice pass: hit flash, shake, damage numbers, pickup burst
- Basit telemetry log
- Ilk 3-5 tester ile gozlemsel test

Teslim:

- Test sonunda tek metrik not edilir:
  - `hemen ikinci run acma davranisi`

## Sprint Sonu Kararlari

- Kontrol iyi ama build zayifsa: guc havuzu yeniden yazilir
- Build iyi ama snake hissi zayifsa: govde savas kimligi derinlestirilir
- Ikisi de zayifsa: konsept sadeletilir veya pivot edilir
