# Retention and Monetization

## Ilke

Core gameplay retention'i tasir. Monetization, retention zarar gormeden oyuna eklenir.

## Retention Katmanlari

### D1

- Kisa onboarding
- Ilk run icinde garantili guclenme
- Run sonu near-miss veya "yeni build denerim" hissi

### D3

- Basit meta ilerleme
- Yeni baslangic relic secenegi
- Yeni power family acilisi

### D7

- Yeni biome
- Yeni dusman tipi
- Daha zor boss paterni
- Build koleksiyonu ve mastery hissi

## Core Retention Hooks

- Near-miss fail
- Acilabilir yeni build yolu
- Run sonu token odulu
- Kisa session icinde tamamlama hissi

## Monetization Sirasi

### Faz 1

- Monetization yok
- Sadece retention dogrulanir

### Faz 2

- Rewarded ad: run sonu bonus chest
- Rewarded ad: opsiyonel baslangic blessing testi

### Faz 3

- Starter pack
- Cosmetic skin
- Meta accelerator

Interstitial, retention sinyali bozulmadan once eklenmez.

## Ilk IAP Prensipleri

- Pay-to-win degil, time-to-variety satilir
- Ilk satin alim ucuz ve dusuk riskli olur
- Kozmetik, kimlik ve koleksiyon fantezisini destekler

## Indirilme Potansiyeli Icin Hook

Store ve creative tarafinda tek mesaj:

`Uzadikca sadece buyumuyorsun; savas alanini govdenle kontrol ediyorsun.`

## Viral Video Anlari

- Cok uzun govde ile arena'yi kapatma
- Tek build sinerjisiyle elite paket eritme
- Boss'u kil payi canla bitirme

## Olculmesi Gereken Erken Sinyaller

- Run sonu ikinci deneme orani
- Ilk 10 dakikada terk orani
- Hangi guclerin en cok secildigi
- Boss'a ulasan oyuncu orani

## Kirmizi Cizgiler

- Reklam oyuncunun olumsuz duygusuna bindirilmez
- Revive, ilk monetization araci olarak kullanilmaz
- Meta ilerleme tamamen satin alima baglanmaz
