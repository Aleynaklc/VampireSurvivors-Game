# Snake Roguelite

Mobil odakli, tek parmakla oynanan, 4-6 dakikalik run'lar sunan bir `snake roguelite`.

Temel vaadi:

`Silahin yok; yilanin govdesi hem kalkan hem silahtir.`

Bu klasor, greenfield oyun projesi icin urun ve teknik baslangic artefaktlarini tutar.

## Belgeler

- `docs/product_brief.md`: Oyun kimligi, hedef kitle, core loop ve farklilasma.
- `docs/mvp_spec.md`: Ilk oynanabilir surumun kapsami ve kabul kriterleri.
- `docs/technical_architecture.md`: Unity tarafi moduller, sistem ayrimi ve performans notlari.
- `docs/sprint_01.md`: Ilk sprintte teslim edilecek islerin sirasi.
- `docs/retention_monetization.md`: Erken retention ve monetization prensipleri.
- `docs/unity_scene_setup.md`: Prototype scene'i Unity'de nasil ayaga kaldiracagin.
- `docs/current_state_map.md`: Oyunun mevcut durumunu, eksikleri ve sonraki yolu gorsel harita olarak gosterir.
- `docs/prototype_power_values.md`: Ilk 5 power icin onerilen test degerleri.
- `docs/prototype_relic_values.md`: Ilk pre-run relic/loadout degerleri.
- `docs/prototype_enemy_values.md`: Chaser, Dasher, Tank ve Boss icin baslangic tuning degerleri.

## Ilk Uretim Sirasi

1. Pure gameplay prototype
2. Snake combat feel
3. Draft-based power system
4. Single run loop with boss
5. Local telemetry for playtest
